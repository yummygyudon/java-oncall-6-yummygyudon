## 📝 Summary
#### ⁜ 배정 규칙 Keyword
- [x] 기본적으로 순번에 따라
- [x] 평일과 휴일(토요일, 일요일, 법정공휴일) 비상 근무 순번을 다르게 운영
  - [x] 평일 순번과 휴일 순번의 순서는 다를 수 있다.
- [x] 평일 순번, 휴일 순번에 각각 1회 편성되어야 한다
- [x] 비상 근무자는 어떤 경우에도 연속 2일은 근무할 수 없다.
  - [ ] 순번상 특정 근무자가 연속 2일 근무하게 되는 경우, **"다음 근무자와 순서"**를 바꿔 편성
    - _수아가 평일인 목요일에 비상 근무를 서고, 다음 날인 금요일이 휴일이면서 순번상 또다시 수아가 근무해야 할 경우,
      다음 휴일 근무자와 순서를 바꿔서 근무_

## 🛠 Feature List
### 🗓 기능
- [x] 기본적으로 입력 순번에 따라 배정한다.
- [x] 평일과 휴일(토요일, 일요일, 법정공휴일) 비상 근무 순번을 각자 별도로 다르게 배정한다.
  - [x] 단, 평일 순번, 휴일 순번에 각각 1회씩만 편성한다.
- [ ] 순번상 특정 근무자가 연속 2일 근무하게 되는 경우, **"다음 근무자와 순서"**를 바꾼다.

<br/>

### 💬 입력
- [x] 월(숫자)과 시작 요일(일, 월, 화, 수, 목, 금, 토) 정보
    - [x] Message : `비상 근무를 배정할 월과 시작 요일을 입력하세요>`
    - [x] `5,월` 형식으로 입력받는다.
    - [x] 연도는 고려하지 않는다
    - [x] 매년 2월은 28일까지만 있다
    - [x] 비상 근무를 배정할 월과 시작 요일의 입력 값이 올바르지 않은 경우, '비상 근무를 배정할 월과 시작 요일'부터 다시 입력받는다.
    - [x] 아닌 경우, `[ERROR] ~`라는 에러 메시지를 보여준다.

<br/>

- [x] 평일 비상 근무 순서
    - [x] Message : `평일 비상 근무 순번대로 사원 닉네임을 입력하세요>`
    - [x] `준팍,도밥,고니,수아,루루,글로,솔로스타,우코,슬링키,참새,도리` 형식으로 입력받는다.
    - [x] 평일 순번의 입력 값이 올바르지 않은 경우, '평일 순번'부터 다시 입력 받는다.
    - [x] 아닌 경우, `[ERROR] ~`라는 에러 메시지를 보여준다.

<br/>

- [x] 휴일(토요일, 일요일, 법정공휴일) 비상 근무 순서
  - [x] Message : `휴일 비상 근무 순번대로 사원 닉네임을 입력하세요>`
  - [x] `수아,루루,글로,솔로스타,우코,슬링키,참새,도리,준팍,도밥,고니` 형식으로 입력받는다.
  - [x] 휴일 순번의 입력 값이 올바르지 않은 경우, '평일 순번'부터 다시 입력 받는다.
  - [x] 아닌 경우, `[ERROR] ~`라는 에러 메시지를 보여준다.


<br/>

## 🚨 Exception Situation 🚨
> 모든 에러 메시지는 `[ERROR]`로 시작하도록 한다.
### Case 1. 잘못된 비상 근무를 배정할 월과 시작 요일을 입력했을 때 :
- [x] 잘못된 월
    - ex. `0,월`, `-1,월`, `13,월`
- [x] 잘못된 요일
    - ex. `1,촤`, `12,귱`, `11,동규`

### Case 2. 잘못된 개수의 비상 근무자 닉네임을 입력했을 때
- [x] 5개 미만
- [x] 35명 초과


### Case 3. 잘못된 형식의 비상 근무자 닉네임을 입력했을 때
- [x] 중복된 닉네임이 있을 경우
- [x] 5자 초과
